其他类型转整形
```python
int('25')
# 25
```
 数字转字符串
```python
str(3820594)
# '3820594'
```


字符串转列表
（数字不能直接转列表）
```python
list('3820594')  
# ['3', '8', '2', '0', '5', '9', '4']
```

 列表转元组
```python
tuple(list('3820594'))
# ('3', '8', '2', '0', '5', '9', '4')
```


 元组转列表
```python
list(tuple('3820594'))
# ['3', '8', '2', '0', '5', '9', '4']
```

元组转字符串
```python
number = ('3', '8', '2', '0', '5', '9', '4')
"".join(number)
# '3820594'
```

列表转字符串
```python
number = ['3', '8', '2', '0', '5', '9', '4']
"".join(number)
# '3820594'
```
