# **正文部分**

正文部分，根据我个人经验，一般有四种类型吧

第一种，一张图都没有，全是字，超级多的字

第二种，图很多，字很少，主要是图片排版

第三种，一般一张图，加上不是很多的字，图和字相当

第四种，以表格、各种图表为主，比如柱状图、折线图、饼图等
